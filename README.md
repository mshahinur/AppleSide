# Apple Side

A static, scroll-driven product landing page for the iPhone 16 Pro.

Built with plain HTML, CSS, and JavaScript, this project includes:
- hero video and scroll-scrubbed content
- 3D model viewer powered by Three.js
- animated reveal sections and parallax effects
- all assets served from the `videos/` folder

## Run locally

From the project folder, start a simple local server:

```bash
python -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000
```

## Deployment

This site is fully static and works well on GitHub Pages.
Vercel is optional, but not required for this project.
